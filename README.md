# Мастер строй
  **Landing page компании "Мастер строй"** — страница магазина строительных материалов и сопутствующих товаров.

## Описание

Одностраничный сайт (landing page), разработанный для компании «Мастер строй». Проект включает:
- полную вёрстку по макету Figma;
- адаптивный дизайн для мобильных устройств;
- шаблонизацию HTML‑элементов с использованием JSON;
- мобильную навигацию.

Цель проекта — представить ассортимент компании и обеспечить удобный пользовательский опыт на любых устройствах.

## Технологии

- **HTML** — разметка страницы;
- **CSS/SCSS** — стилизация и препроцессинг;
- **JavaScript** — интерактивные элементы;
- **Gulp** — сборка проекта, автоматизация задач.

## Реализованные функции

- Вёрстка по макету Figma (с соблюдением типографики, отступов, цветов).
- Адаптивный дизайн:
  - полная адаптация под мобильные устройства (смартфоны, планшеты);
  - гибкие сетки и медиазапросы.
- Шаблонизация HTML‑элементов:
  - использование JSON для динамической генерации повторяющихся блоков;
  - упрощение поддержки и масштабирования разметки.
- Мобильная навигация:
  - выпадающее меню для маленьких экранов;
  - удобное взаимодействие с элементами.
